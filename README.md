Lead Qualification Tool

A Python-based Lead Qualification Tool that simulates a real-world pre-sales workflow by evaluating customer information and recommending the most appropriate solution based on business and technical factors.

This project was built as part of my journey toward becoming a Sales Engineer and Solutions Engineer, combining Python programming with customer discovery and solution recommendation.

Overview

The application collects customer information and calculates a lead score using several qualification criteria:

Company Size
Estimated Budget
Business Pain Point
Project Urgency
Technical Fit

Based on the calculated score, the program recommends:

A suitable solution plan
The next best sales action
An overall lead qualification score
